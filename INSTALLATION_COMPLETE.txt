🚀 NEXT STEPS
================================================================================

1. SET UP YOUR API KEY
   ------------------
   a) Get Claude API key: https://console.anthropic.com/
   b) Copy .env.example to .env:
      $ cp .env.example .env
   c) Edit .env and add your key:
      ANTHROPIC_API_KEY=sk-ant-api03-your-key-here

2. RUN THE AGENT
   --------------
   $ source .venv/bin/activate
   $ python agent.py

   You should see:
   INFO: Trip Planner Agent started!
   INFO: Agent address: agent1q...
   INFO: Chat Protocol enabled

3. TEST LOCALLY
   ------------
   $ python test_local.py

   This will show example itineraries and create calendar files.

4. DEPLOY TO AGENTVERSE
   --------------------
   See DEPLOYMENT.md for complete guide:
   - Register at agentverse.ai
   - Upload code or connect GitHub
   - Set ANTHROPIC_API_KEY environment variable
   - Enable Chat Protocol
   - Publish to ASI:One
